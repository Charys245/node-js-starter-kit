# node-js-starter-kit
Un starter kit moderne et prêt à l'emploi pour vos projets Node.js/Express  Ce starter kit est né de ma propre expérience d'apprentissage. Il reflète les bonnes pratiques que j'ai acquises lors de ma certification Node.js/Express et les structure dans un format réutilisable. Plus qu'un simple template, c'est une base de code commentée et évolutive.
